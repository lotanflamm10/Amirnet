FILES INCLUDED
1. questions_extended.json -> full questions file with new hard questions already added
2. vocab_extended.json -> full vocab file with new hard vocabulary already added
3. hard_questions_addon.json -> only the new hard questions pack
4. hard_vocab_addon.json -> only the new hard vocabulary pack

HOW TO USE
Option A
Replace your existing questions.json with questions_extended.json
Replace your existing vocab.json with vocab_extended.json

Option B
Manually copy the new items from hard_questions_addon.json into the matching arrays inside questions.json
Copy the new items from hard_vocab_addon.json into vocab.json

NOTES
The generated questions focus on hard Amiram-style practice.
